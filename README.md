# RepFlow

Android MVP на Kotlin + Jetpack Compose для тренировок по подтягиваниям и отжиманиям.

## Что уже есть
- стартовый тест для подтягиваний и отжиманий;
- генерация 6-недельного плана с контрольными тестами;
- пошаговое ведение тренировки с паузами между подходами;
- ввод фактических повторов/секунд;
- адаптивная пересборка оставшегося плана после тренировки;
- локальное хранение через Room;
- экраны плана, истории и техники;
- unit-тесты на оценку и планировщик.

## Как открыть
1. Открой папку проекта в Android Studio.
2. Дождись Gradle Sync.
3. При необходимости сгенерируй Gradle Wrapper из IDE или установленного локально Gradle.
4. Запусти приложение на Android-эмуляторе или устройстве.

## GitHub И Релизы
- CI собирает debug APK и гоняет unit-тесты на push и pull request.
- Релиз создается по тегу формата `vX.Y.Z`.
- Версия приложения подставляется через Gradle properties `appVersionName` и `appVersionCode`.
- Правила для коммитов, версионирования и пуша описаны в [AGENTS.md](/home/vase/Projects/repflow/AGENTS.md) и [CONTRIBUTING.md](/home/vase/Projects/repflow/CONTRIBUTING.md).

## Структура
- `app/src/main/java/com/vaslit/domain` — правила оценки и планировщик.
- `app/src/main/java/com/vaslit/data` — Room и репозиторий.
- `app/src/main/java/com/vaslit/ui` — Compose UI и ViewModel.

## Ограничения текущей реализации
- только офлайн-режим;
- без аккаунта, облака, уведомлений и Garmin;
- предустановленные программы без пользовательского конструктора.
